# nlauncher

My personal launcher built with GPUI.

## Features

- Fast application launcher with fuzzy search
- Calculator integration
- Modern UI with Wayland support

## Sub-projects

- **nwidgets**: GPUI-based widgets and system services (notifications, audio, network, etc.)
- **ndownloader**: Download manager
- **Loungy**: Additional launcher components
- **zed**: Zed editor integration

## Development

```bash
nix develop
cargo run
```
